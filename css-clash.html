CSS Tips & Tricks

Boost your CSS skills with these helpful tips and techniques.

Perfect Centering

The flexbox way to center elements

Centering elements both horizontally and vertically has historically been a challenge in CSS. With flexbox, it's now simple and reliable.

.parent {
/* The key is to use display: flex */
display: flex;
justify-content: center; /* Centers horizontally */
align-items: center; /* Centers vertically */
height: 300px; /* Make sure the parent has a height */
}

CSS Shape Tricks

Create complex shapes with pure CSS

You can create many complex shapes using just CSS. Here's how to create a simple heart shape:

.heart {
width: 100px;
height: 90px;
position: relative;
}

.heart:before, .heart:after {
content: "";
position: absolute;
top: 0;
width: 50px;
height: 80px;
border-radius: 50px 50px 0 0;
background: red;
}

.heart:before {
left: 50px;
transform: rotate(-45deg);
transform-origin: 0 100%;
}

.heart:after {
left: 0;
transform: rotate(45deg);
transform-origin: 100% 100%;
}

CSS Variables (Custom Properties)

Make your code more maintainable

CSS Variables (also known as Custom Properties) allow you to store reusable values in one place and reference them throughout your stylesheet.

/* Define variables in the :root selector */
:root {
--primary-color: #9b6dff;
--secondary-color: #ff6b9d;
--spacing-unit: 8px;
--border-radius: 4px;
}

/* Use variables with var() function */
.button {
background-color: var(--primary-color);
padding: calc(var(--spacing-unit) * 2);
border-radius: var(--border-radius);
}
