Box Model

width

The width property sets the width of an element's content area. It can be applied to most HTML elements except non-replaced inline elements, table rows, and row groups.

Syntax

width: auto | length | percentage | max-content | min-content | fit-content | fit-content(length) | initial | inherit;

The width property can accept various types of values, such as lengths (px, em, rem), percentages, or keywords.

Values

Value Description
auto The browser will calculate and select a width for the specified element (default).
length Defines the width in px, cm, em, etc. Learn more about length units.
percentage Defines the width as a percentage of the containing block's width.
max-content The intrinsic preferred width.
min-content The intrinsic minimum width.
fit-content The larger of min-content and the smaller of max-content and available width.
initial Sets this property to its default value.
inherit Inherits this property from its parent element.

Examples

Basic width example Result
.box-1 { width: 100px; height: 100px; background-color: #9b6dff; } .box-2 { width: 200px; height: 100px; background-color: #ff6d9b; } .box-3 { width: 300px; height: 100px; background-color: #6d9bff; }
Percentage width example Result
.container { width: 100%; background-color: #f3eaff; padding: 10px; } .box-percentage { width: 50%; height: 100px; background-color: #9b6dff; }

Important Note

The width property does not include padding, borders, or margins. It sets the width of the content area only. If you want the element to maintain a specific width no matter what, you can use the box-sizing property with the value border-box.
